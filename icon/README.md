# Icon assets

The source artwork for the BarTranslateACO app icon lives in `src/BarTranslate.icon`.

`appicon-exports/` contains the default exported PNG sizes that are copied into `BarTranslate/Assets.xcassets/AppIcon.appiconset`. Other experimental light, dark, clear, and tinted export variants were removed to keep the repository focused on the icon set that is actually used by the app.
